Java If...Else

The if Statement

The if statement runs a specific block of code if its condition is true.

if (condition) {
// block of code
}

The else Statement

The else statement runs a specific block of code if the previous if statement's condition is false.

if (condition) {
// block of code if true
} else {
// block of code if false
}

The else if Statement

The else if statement runs a specific block of code if its condition is true, and all other previous if and else if statements' conditions are false.

if (condition 1) {
// block of code if condition 1 is true
} else if (condition 2) {
// block of code if condition 1 is false, and condition 2 is true
}

  • There is a short-hand way to write if-else statements, known as a ternary operator.
    • condition ? code block if true : code block if false
Java Switch

Java Switch Statements

The switch statement runs a specific code block of code depending on which case the expression triggers.

switch (expression) {
case x:
// code block
break;
case y:
// code block
break;
default:
// code block if no cases are met
}

The break Keyword

The break keyword ends and skips a block of code (e.g. switch statements).


Java While Loops

Java While Loop

The while loop runs a block of code repeatedley as long as the condition is true.

while (condition) {
// code block
}

The Do/While Loop

The do/while loop is a variation of a while loop that runs a block of code at least once, even if the if condition is initially false.

do {
// code block
}
while (condition);
Java For Loops

Java For Loop

The for loop runs a block of code a set number of times.

for (statement 1; statement 2; statement 3) {
// code block
}
  • Statement 1 executes only once when the for loop starts.
  • Statement 2 defines how many times a block of code runs.
  • Statement 3 executes after every time the block of code executes.

The For-Each loop

The for-each loop runs a block of code on every element in an array.

for (type variableName : arrayName) {
// code block
}
Java Continue

Java Continue Statement

The continue statement ends a block of code.

It differs from the break statement in that subsequent loops will run for continue, but be skipped for break.

for (int i = 0; i < 5; i++) {
if (i === 1) {
continue;
}
}
